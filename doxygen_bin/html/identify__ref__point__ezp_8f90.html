subroutine identify_ref_point_dble ( double precision, intent(in)  col_spc,
double precision, intent(inout)  col_ref,
integer, intent(in)  decomp_id 
)

The EZ_PARALLEL reference point identification subroutine for DOUBLE PRECISION column spacing and column reference point. Assuming a linear spacing of grid points, adjusts the physical position of the reference point in the physical direction corresponding to the second dimension of the grid.

Author
Jason Turner, University of Wisconsin-Madison
Parameters
[in]col_spcPhysical spacing of columns in the grid, e.g., if the second dimension of the grid represents the z-direction, then col_spc corresponds to the spacing of grid points in the z-direction.
[in,out]col_refPhysical position of the reference point in the second dimension of the grid, e.g., the z-coordinate.
[in]decomp_idID number of the grid decomposition, between 1 and the number of "unique" grid decompositions.
